Documentation for integrating the PIM system with external platforms.

## Available Documentation

### ⭐ [Multichannel Sync Guide](MULTICHANNEL_SYNC.md)

Complete guide to syncing PIM products across multiple marketplaces and storefronts.

**Supported Channels:**
- **Solr** - Search index for fast product discovery
- **eBay** - International marketplace
- **Amazon** - Italy marketplace (configurable)
- **Trovaprezzi** - Italian price comparison
- **ManoMano** - DIY & home improvement marketplace
- **B2B** - Wholesale/business storefront
- **B2C** - Retail/consumer storefront

**Features:**
- ✅ Multi-channel sync (7 channels)
- ✅ Batch tracking across all channels
- ✅ Channel-specific metadata (tenant_id, store_id)
- ✅ Flexible single or multiple store support
- ✅ Adapter pattern for easy extensibility

[→ Read the full guide](MULTICHANNEL_SYNC.md)
