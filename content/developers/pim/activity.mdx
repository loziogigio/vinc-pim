# Activity

The **Attività** section surfaces the tenant's background-job feed:
imports, bulk updates, AI enrichments, search reindexes, and media
cleanups all appear here. Every long-running PIM operation returns a{" "}
<code>job_id</code> that you can look up through this API.

## Endpoints

<Endpoint method="GET" path="/api/b2b/pim/jobs" />
<Endpoint method="GET" path="/api/b2b/pim/jobs/:job_id" />
<Endpoint method="GET" path="/api/b2b/pim/jobs/:job_id/events" />
<Endpoint method="POST" path="/api/b2b/pim/jobs/:job_id/cancel" />
<Endpoint method="POST" path="/api/b2b/pim/jobs/:job_id/retry" />
